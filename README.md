# Shehryal First

Shehryal First is a desktop creator utility for securely authorizing TikTok and uploading or directly publishing selected videos through TikTok's official Content Posting API.

## How it works

1. Connect your TikTok account using TikTok authorization.
2. Select a video stored on your computer.
3. Review the post settings.
4. Upload a draft or directly publish after confirmation.

The app does not ask for your TikTok password. Access can be revoked from TikTok account settings.
